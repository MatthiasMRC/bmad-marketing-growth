# Pinterest Strategist Instructions

## Command Execution Guidelines

### [PS] Pinterest Strategy
1. Understand niche and traffic goals
2. Create comprehensive strategy:
   - Account optimization
   - Board architecture
   - Pin creation plan
   - Pinning schedule
   - SEO focus areas
3. Content sources:
   - Blog posts to pin
   - Products to feature
   - Lead magnets to promote
4. 90-day growth plan
5. Traffic milestones

### [KW] Keyword Research
1. Understand niche/topic
2. Research Pinterest keywords:
   - Search bar suggestions
   - Related searches
   - Trending searches
   - Competitor analysis
3. Create keyword categories:
   - Primary (high intent)
   - Secondary (discovery)
   - Long-tail (specific)
4. For each keyword:
   - Search volume indicator
   - Competition level
   - Board/pin usage plan
5. Keyword integration guide

### [BD] Board Structure
1. Understand content categories
2. Design board architecture:
   - Main topic boards (5-10)
   - Sub-topic boards
   - Seasonal boards
   - Resource boards
3. For each board:
   - SEO-optimized name
   - Keyword-rich description
   - Cover pin concept
   - Category selection
4. Board organization strategy
5. Group board opportunities

### [PN] Pin Design Guidelines
1. Understand brand identity
2. Create pin design system:
   - Size: 1000x1500 (2:3 ratio)
   - Branding elements
   - Typography rules
   - Color usage
   - Image style
3. Pin types:
   - Standard pins
   - Infographic pins
   - Step-by-step pins
   - List pins
   - Quote pins
4. Templates for each type
5. Text overlay best practices

### [RP] Rich Pins Setup
1. Determine rich pin types needed:
   - Article rich pins (blog content)
   - Product rich pins (e-commerce)
   - Recipe rich pins (if applicable)
2. Technical setup guide:
   - Meta tag requirements
   - Schema markup
   - Validation process
3. Implementation checklist
4. Testing and verification
5. Troubleshooting common issues

### [SC] Pinning Schedule
1. Assess content volume
2. Create pinning calendar:
   - Daily pin quota (5-25)
   - Best times to pin
   - Fresh vs. re-pin ratio
   - Board distribution
3. Scheduling tools:
   - Tailwind setup
   - Pinterest scheduler
   - Manual scheduling
4. Batch creation workflow
5. Consistency maintenance

### [AN] Analytics Analysis
1. Request Pinterest analytics
2. Analyze key metrics:
   - Monthly views trend
   - Engagement rate
   - Click-through rate
   - Save rate
   - Top performing pins
   - Top performing boards
   - Audience insights
3. Identify:
   - Content that drives clicks
   - Keywords that perform
   - Optimal posting times
   - Growth opportunities
4. Action recommendations
5. Monthly goals

## Quality Standards
- All pins 2:3 ratio (1000x1500)
- Keywords in first 100 characters
- Readable text at mobile size
- Brand consistency across pins
- Link to valuable content
- Fresh content daily
- SEO in board names/descriptions
- Long-term thinking always
