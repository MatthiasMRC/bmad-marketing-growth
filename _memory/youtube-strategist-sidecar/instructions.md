# YouTube Strategist Instructions

## Command Execution Guidelines

### [VS] Video Strategy
1. Understand channel goals and niche
2. Create comprehensive strategy:
   - Content pillars (3-5 categories)
   - Video formats mix (tutorials, vlogs, reviews, etc.)
   - Posting schedule
   - Growth milestones
3. For each content pillar:
   - 10 video ideas
   - Target keywords
   - Thumbnail style
   - Success metrics
4. 90-day content calendar

### [VO] Video Optimization
1. Get video topic/script
2. Optimize:
   - Title (5 variations, keyword-first)
   - Description (first 150 chars crucial, full SEO)
   - Tags (primary + secondary keywords)
   - Cards & end screens strategy
   - Timestamps/chapters
3. Provide:
   - SEO analysis
   - Competitor title comparison
   - Search volume data if available

### [TN] Thumbnail Brief
1. Understand video topic
2. Create thumbnail brief:
   - Visual concept (what to show)
   - Text overlay (max 3-4 words)
   - Color scheme
   - Emotion to convey
   - Face expression (if applicable)
3. Provide 3 concept variations
4. Reference successful thumbnails in niche
5. Technical specs (1280x720, contrast, mobile-visible)

### [SC] Script Structure
1. Understand video topic and length
2. Create script outline:
   - Hook (first 30 seconds - critical)
   - Pattern interrupt moments
   - Value delivery sections
   - Engagement prompts (subscribe, comment)
   - CTA placement
   - End screen lead-in
3. Include:
   - Timing for each section
   - B-roll suggestions
   - Retention optimization tips
4. Provide hook variations

### [SH] Shorts Strategy
1. Analyze channel and goals
2. Create Shorts plan:
   - Content types that work in niche
   - Repurposing from long-form
   - Original Shorts concepts
   - Posting frequency (daily ideal)
3. For each Short:
   - Hook in first 1 second
   - Duration sweet spot
   - Text overlay strategy
   - Music/sound selection
4. Cross-promotion with long-form

### [CA] Channel Audit
1. Request channel access/info
2. Audit checklist:
   - Channel branding (banner, profile, trailer)
   - Metadata (about, links)
   - Content analysis (top/bottom performers)
   - SEO health check
   - Thumbnail consistency
   - Publishing consistency
   - Playlist organization
   - Community engagement
3. Provide:
   - Strengths
   - Weaknesses
   - Quick wins
   - Long-term improvements
4. Priority action list

### [PL] Playlist Strategy
1. Review channel content
2. Create playlist architecture:
   - Series playlists (learning paths)
   - Topic playlists (evergreen)
   - Best of playlist
3. For each playlist:
   - SEO-optimized title
   - Description with keywords
   - Video order strategy
   - Custom thumbnail concept
4. Homepage layout recommendations

## Quality Standards
- All titles < 60 characters
- Descriptions: keywords in first 150 chars
- Thumbnails: readable at mobile size
- Scripts: hook in first 10 seconds
- Tags: mix of broad and specific
- Always think search intent
