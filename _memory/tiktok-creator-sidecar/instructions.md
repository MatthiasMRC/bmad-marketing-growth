# TikTok Creator Instructions

## Command Execution Guidelines

### [TS] TikTok Strategy
1. Understand niche and goals
2. Create comprehensive strategy:
   - Content pillars (3-4)
   - Video formats to use
   - Posting frequency (daily ideal)
   - Engagement tactics
   - Growth timeline
3. For each content pillar:
   - 10 video ideas
   - Format variations
   - Trend opportunities
4. First 30 days action plan
5. Metrics to track

### [TH] Trend Hacking
1. Research current trends:
   - Trending sounds
   - Trending formats
   - Trending hashtags
   - Creator trends
2. For each relevant trend:
   - How to adapt for niche
   - Script concept
   - Timing (trend lifecycle)
   - Risk assessment
3. Create 5 trend-based video concepts
4. Trend monitoring strategy

### [HK] Hook Creation
1. Understand topic/message
2. Create hook variations:
   - Controversial statement
   - Unexpected fact
   - Direct question
   - Pattern interrupt
   - "Wait for it" tease
   - POV hook
   - "You need to know this"
3. For each hook:
   - First second visual
   - Opening line
   - Why it works
4. Provide 10 hook variations

### [SC] Script Optimization
1. Understand video concept
2. Create TikTok script:
   - Hook (0-1 sec): Stop the scroll
   - Setup (1-5 sec): Context
   - Value (5-30 sec): Main content
   - Payoff (last 3 sec): CTA or loop
3. Include:
   - Text overlay copy
   - B-roll suggestions
   - Sound recommendation
   - Hashtags
4. Variations for different lengths

### [DU] Duet/Stitch Strategy
1. Research opportunity types:
   - Viral videos to react to
   - Competitors to engage with
   - Questions to answer
   - Debates to join
2. Create engagement plan:
   - Target videos to duet/stitch
   - Response angle
   - Value to add
   - Timing strategy
3. Weekly engagement quota
4. Best practices:
   - Add genuine value
   - Don't be negative
   - Bring unique perspective

### [PS] Posting Schedule
1. Analyze target audience:
   - Demographics
   - Active times
   - Content consumption patterns
2. Create posting calendar:
   - Optimal posting times
   - Content type rotation
   - Testing schedule
   - Trend response slots
3. Frequency recommendations:
   - Minimum viable: 1/day
   - Optimal: 3-5/day
   - Maximum: 7-10/day
4. Batch creation strategy

### [AN] Analytics Analysis
1. Request metrics data
2. Analyze:
   - Views by video type
   - Watch time patterns
   - Engagement rates
   - Follower growth
   - Best performing content
   - Traffic sources
3. Identify:
   - What's working
   - What's not working
   - Opportunities
   - Threats
4. Action recommendations
5. A/B test suggestions

## Quality Standards
- Hook in first 1 second or lose them
- Keep videos 15-60 seconds optimal
- Vertical format always (9:16)
- Native content looks > polished
- Post volume matters
- Reply to comments fast
- Use trending sounds when possible
- Text overlays for silent viewing
